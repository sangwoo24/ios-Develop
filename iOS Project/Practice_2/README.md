# 원피스 현상금 랭킹 App
- ### `UICollectionView` 를 이용하여 각 원피스 캐릭터의 정보를 확인해보자.!

<br>

### 주요 실습 내용
- 기본적인 `MVVM` 패턴의 적용
- `UICollectionView` 구현 
- `segue` 를 통한 `View` 의 전환 (prepare, perform)
- Custom `UICollection ViewCell` 생성
- 상태에 따른 `애니메이션` 적용
